# Order Table Analysis 📦

A business data analysis project examining order records to identify trends in sales, delivery, and customer behavior using Power BI.

## 🛠️ Tools Used
![Power BI](https://img.shields.io/badge/PowerBI-F2C811?style=flat&logo=powerbi&logoColor=black)
![Excel](https://img.shields.io/badge/Excel-217346?style=flat&logo=microsoft-excel&logoColor=white)

## 📊 Objective
To analyze order data and generate insights that help businesses optimize their operations and improve customer experience.

## 🔍 Key Areas Explored
- Order volume and revenue trends over time
- Top products and categories by sales
- Delivery performance and fulfillment analysis
- Customer segmentation based on order history
